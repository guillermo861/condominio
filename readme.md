## Descripción

Este es un proyecto demostrativo de un posible uso de Laravel y Vue de forma natural

## Instalación

Se recomienda el uso de Docker para agilizar le proceso.

- Usando Docker: docker run --name mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=root -d mysql:5.7
- Sin docker instalar Mysql
- Descargar el repositorio
